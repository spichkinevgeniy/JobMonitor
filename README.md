# JobMonitor

JobMonitor - Telegram-сервис для автоматического мониторинга IT-вакансий и персонального отбора под профиль кандидата.
Проект снижает шум в вакансиях: собирает сообщения из каналов, извлекает структуру и отправляет только релевантные предложения.

**Статус проекта:** Active development, production-ready beta  
**Ссылка на бота:** [Открыть бота](https://t.me/your_bot_username)

## Что это

JobMonitor помогает кандидатам в IT не просматривать вручную десятки Telegram-каналов.

### Для кого
- разработчики и инженеры, ищущие вакансии через Telegram;
- кандидаты, которым нужен персональный поток релевантных предложений.

### Ключевые возможности
- сбор вакансий из Telegram-каналов;
- определение, является ли сообщение вакансией;
- AI-парсинг вакансии и профиля кандидата;
- фильтрация по профилю и настройкам пользователя;
- доставка подходящих вакансий через бота.

### Ограничение текущей версии
- профиль кандидата передается через резюме в формате PDF.

## Как это работает

1. **Передача профиля** - пользователь отправляет резюме (PDF).
2. **Извлечение структуры** - AI выделяет специализацию, язык, стек, опыт, зарплату и формат работы.
3. **Отбор и доставка** - базовый отбор идет по профилю, доп. фильтры применяются только при включении пользователем.

## Прозрачность отбора

- предфильтр по пересечению специализаций и основных языков;
- фильтр по опыту при включенном пороге;
- фильтр по зарплате в строгом режиме и при сопоставимой валюте;
- фильтр по формату работы (soft/strict), который пользователь настраивает в профиле.

## Архитектура и поток данных

1. **Источники** - Telethon читает сообщения из заданных каналов.
2. **Mirror-канал** - сообщения пересылаются в служебный канал для стабильной обработки.
3. **AI-парсинг** - вакансии классифицируются и структурируются, резюме превращается в профиль поиска.
4. **Хранение и матчинг** - данные сохраняются в PostgreSQL, сервис матчинга выполняет предфильтр и доменные проверки.
5. **Доставка** - aiogram-бот отправляет релевантные вакансии пользователям.

## Надежность и наблюдаемость

- **Sentry** - сбор ошибок;
- **Prometheus** - метрики процесса и конвейера вакансий;
- **Logfire** - трассировка AI-вызовов.

## Технологический стек

- Python 3.12
- aiogram
- Telethon
- SQLAlchemy + asyncpg
- PostgreSQL
- pydantic-ai + Google model provider
- Alembic
- Prometheus / Sentry / Logfire
